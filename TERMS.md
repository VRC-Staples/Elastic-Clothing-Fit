## Auto-Updater Network Notice

This add-on contacts the GitHub releases API each time it loads in order to
check whether a newer version is available.

- The only information transmitted is an HTTP request to:
  `https://api.github.com/repos/VRC-Staples/Elastic-Clothing-Fit/releases/latest`
  with the User-Agent header `ElasticClothingFit-Updater`
- No personal data, identifiers, or usage information is sent
- No analytics, tracking, or telemetry of any kind is collected
- The full network code is available for inspection in `elastic_fit/updater.py`
